# 45 Days Machine Learning & Deep Learning Journey 🚀

This repository contains my **45 days learning journey of Machine Learning, Deep Learning, and Generative AI**.

The goal of this challenge is to study important concepts of ML and DL step by step and also implement small projects or experiments for better understanding.

During these 45 days I will cover topics starting from **data preprocessing and classical machine learning models**, then move towards **deep learning, NLP, and modern AI systems like LLMs and RAG**.

I believe learning by building small experiments is the best way to understand AI concepts.

---

## What this journey includes 📚

In these 45 days I will explore:

- Data loading and preprocessing
- Exploratory data analysis
- Feature engineering and feature selection
- Regression models

- Clustering algorithms
- Ensemble learning methods
- Model evaluation techniques
- Hyperparameter tuning
- Neural networks
- Convolutional neural networks

- Graph neural networks
- Natural language processing

- Embeddings and vector search
- Retrieval augmented generation
